# 🏥 Hospital ER Analytics Dashboard

## 📊Dataset

Source: Hospital Emergency Room Dataset (public dataset)
The dataset contains anonymized patient visit records used for educational and portfolio purposes.

## 📌 Project Overview

This project is an interactive **Power BI dashboard** built to analyze
the performance of a Hospital Emergency Room (ER). The dashboard helps
stakeholders monitor patient volume, waiting times, referrals,
demographics, and department performance.

## 🎯 Business Objectives

-   Monitor ER performance
-   Analyze patient demographics
-   Measure waiting times
-   Evaluate department referrals
-   Track patient satisfaction
-   Generate actionable insights

## 🛠 Tools Used

-   Power BI Desktop
-   Power Query
-   DAX
-   Microsoft Excel

## 📂 Dataset

Fields include: - Patient ID - Date - Age / Age Group - Gender - Race -
Wait Time - Satisfaction Score - Department Referral - Patient Admission
Flag

# 📈 Key Insights

-   General Practice receives the highest referrals.
-   Neurology has the highest average waiting time.
-   Gastroenterology has the highest patient satisfaction.
-   Many patients wait more than 30 minutes.

# 💡 Recommendations

-   Improve staffing in high-wait departments.
-   Monitor referral trends.
-   Track satisfaction regularly.
-   Reduce average waiting time.
